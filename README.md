## Test task about allocation passengers between flights

### Backend part:

***For the running of the backend part you should use this steps:***

**steps:**
> 1. cd rubiq_backend
> 2. yarn install
> 3. yarn start

***For the running of the frontend part you should use this steps:***

**steps:**
> 1. cd rubiq_frontend
> 2. yarn install
> 3. yarn start
